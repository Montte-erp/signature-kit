# @signature-kit/clicksign

Alchemy v2 remote-signature provider for Clicksign document creation, lookup, cancellation, deletion, and signed-document download.

## Install

```sh
bun add @signature-kit/clicksign effect alchemy
```

## Exports

- `@signature-kit/clicksign`

## Runtime model

SignatureKit packages are Effect-native. Public APIs return typed `Effect.Effect` values; recoverable faults stay in the typed error channel; callers provide required services and layers explicitly at the application boundary.
