---
title: ZapSign
description: "Crie documentos no ZapSign com createZapSignSignatureRequest(...), um token de API Redacted e o endpoint REST oficial."
---

O ZapSign é um provedor remoto que conversa diretamente com `POST /api/v1/docs/`, sem nenhum gateway intermediário. Instale o pacote e crie o documento em dois passos.

<Steps>
  <Step title="Instalar">
    Adicione o runtime principal e o pacote do provedor ZapSign.

    ```package-install
    @signature-kit/core @signature-kit/zapsign
    ```
  </Step>

  <Step title="Crie o documento">
    A função envia `base64_pdf`, os signatários e a autenticação `Authorization: Bearer`. A resposta normaliza `token`, `status` e a primeira `sign_url` em um `RemoteSignatureRequest`.

    ```ts title="zapsign.ts"
    import { createZapSignSignatureRequest } from "@signature-kit/zapsign"
    import { signatureHttpClientLive } from "@signature-kit/core/http"
    import { Effect, Redacted } from "effect"

    const request = yield* createZapSignSignatureRequest(
      {
        apiToken: Redacted.make(process.env.ZAPSIGN_API_TOKEN ?? ""),
        environment: "sandbox",
        locale: "pt-br",
      },
      {
        title: "Contract",
        message: "Sign to complete.",
        documents: [{ fileName: "contract.pdf", mimeType: "application/pdf", content: pdfBytes }],
        recipients: [{ name: "Davi Rocha", email: "davi@example.com", role: "signer" }],
      },
    ).pipe(Effect.provide(signatureHttpClientLive))

    // request.provider === "zapsign"
    // request.signingUrl holds the first signing link returned by ZapSign
    ```
  </Step>
</Steps>

## Limite da API

A implementação atual aceita um único PDF por requisição, alinhada ao fluxo de upload documentado pelo ZapSign.

<Callout type="warn" title="Um PDF por requisição">
  Vários documentos ou qualquer outro tipo MIME falham como `signature-kit.UNSUPPORTED_OPERATION`.
</Callout>

## Erros

Falhas de entrada, de HTTP e de formato da resposta usam `SignatureKitError`, com `provider: "zapsign"` e metadados estruturados quando disponíveis.
