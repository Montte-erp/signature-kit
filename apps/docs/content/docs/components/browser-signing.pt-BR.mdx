## Fluxo mĂ­nimo

```ts title="a1-browser-signing.ts"
import { a1SignaturesLayer } from "@signature-kit/a1/signer"
import {
  createBrowserPdfSignatureBuilderState,
  readBrowserFileBytes,
  signBrowserPdf,
} from "@signature-kit/react/browser-pdf"
import { Effect, Redacted } from "effect"

const bytes = await Effect.runPromise(readBrowserFileBytes(file))

const state = await Effect.runPromise(
  createBrowserPdfSignatureBuilderState({
    id: "contract-a1",
    name: file.name,
    documentId: "contract",
    documentName: file.name,
    pdf: bytes,
    role: { id: "signer", label: "Signer A1", required: true },
    draft: { id: "sig", type: "signature", roleId: "signer", width: 156, height: 42 },
    autoPlacement: { page: "last", slot: "bottom-right", margin: 24 },
  }),
)

const signed = await Effect.runPromise(
  signBrowserPdf({
    pdf: bytes,
    template: state.template,
    fieldId: "sig",
    reason: "Signed locally with A1",
    signatureLength: 16384,
    policy: "pades-ades",
  }).pipe(
    Effect.provide(a1SignaturesLayer({ pfx, password: Redacted.make(password) })),
  ),
)
```

## API

### `readBrowserFileBytes`

```ts
readBrowserFileBytes(file: Blob): Effect.Effect<Uint8Array, ReactIntegrationError>
```

LĂŞ um arquivo/blob do navegador como bytes. Falha com `react.FILE_READ_FAILED`.

### `createBrowserPdfSignatureBuilderState`

```ts
createBrowserPdfSignatureBuilderState(input): Effect.Effect<ReactSignatureBuilderState, ReactIntegrationError>
```

Carrega o PDF com `pdf-lib`, cria um template com um signer role e opcionalmente posiciona a assinatura. Passe `placement` ou `autoPlacement`, nunca os dois.

### `signBrowserPdf`

```ts

```

### `signBrowserPdfBatch`

